## About The Project

The goal of this project is to develop knowledge and learn a simple CRM web application using the Django framework, by following an online Django guide. The app mimics Users and Agents, just how the real world CRM systems work. 

## Usage

- Load into Python env
- Run `python3 manage.py runserver` to start the server.
